# discovery data

`sponsor-register.csv` — the UK government Register of Licensed Sponsors (Worker
and Temporary Worker), used by `scout.py` for the visa hard-gate. It is **large
and regenerable, so it is git-ignored** (see the repo `.gitignore`).

Refresh: download "Register of licensed sponsors: workers" (CSV) from
<https://www.gov.uk/government/publications/register-of-licensed-sponsors-workers>
and save it here as `sponsor-register.csv`.
