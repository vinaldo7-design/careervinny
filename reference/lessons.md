# Lessons

## L001 - 2026-06-18 - enablement-is-not-powerlessness
trigger: JPMorgan "Strategy Senior Associate - AI Enablement" (London, sponsors, AI strategy) - rubric would have applied the deck-production penalty; I said I'd happily do this work and see it as a real opportunity.
type: anti-fit-refinement
target: section 2b - was "Deck-production / no-judgment", now "No-agency / powerlessness"
signal: the rubric used deliverable-type (makes decks / enablement materials) as a proxy for "no judgment". That proxy is wrong. The aversion from the north-star is to POWERLESSNESS - judgment treated as throughput, puppet for a senior exec - not to producing materials. Enablement work at a serious firm often gives judgment MORE room (shapes roadmap, briefs leadership, decides the client message).
delta: re-key section 2b penalty from deliverable-type to agency. Decks/training/enablement = neutral. Penalise only fixed-plan execution with no shaping room.
confidence: low
watch: does this relaxation recur on NON-prestige enablement roles, or only fire when the firm is a name I want? If prestige-only, the true rule is "I'll tolerate low-agency work for a name" - a different, more honest rule.
status: accepted

## L004 - 2026-06-19 - prestige-is-a-multiplier-not-a-gate
trigger: Capco "Data & AI Strategist" + Accenture "Tech Strategy Manager" - both fit hard on the things Vinay said matter most (strategy-setting work, frontier-AI, his ESG×AI edge, client-facing). Cited Lloyds and Capco as great despite not being marquee names.
type: gate-demotion (near-gate → multiplier)
target: §3 NEAR-GATE prestige; §5 bonus multipliers; D019 bands
signal: north-star wrote prestige as "near-mandatory, terminal." Revealed: prestige was a PROXY for the real want, which is STABILITY + SIZE + JOB SECURITY. The proxy over-rejected high-fit roles at successful non-marquee firms. The oracle-trap / 41-yr-test / status-compounds reasoning still holds - so prestige is not deleted, it's demoted to a positive weight.
delta:
  - DEMOTE prestige from near-gate (−30) to a POSITIVE MULTIPLIER/booster. A marquee name lifts a role; its absence no longer sinks one.
  - INSTALL the real gate prestige was standing in for: FIRM STABILITY + SIZE + JOB SECURITY. Successful, large enough to be secure = pass. Sub-Series-C / shaky / too-small still fails (this gate keeps the old anti-fit's protective job).
  - ROUTING: high-prestige + high-fit + low-reachability roles surface as MOONSHOT/networking targets (D019), never gated out. Prestige is a multiplier AND a band-router, never a kill.
confidence: high - two strong consistent real reactions, and it resolves a known proxy (prestige-for-security) the north-star half-acknowledged.
status: accepted
